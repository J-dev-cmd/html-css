Usando tags em HTML

Em HTML podemos usar várias tags para representar vários elementos de uma página. Coloque o mouse sobre cada termo da lista a seguir, para ver as respectivas tags HTML:

  • Listas Ordenadas
  • Listas Não Ordenadas
  • Links
  • Parágrafos
  • Títulos
  • Quebras de linha
  • Linha Horizontal
  • Imagens
  • Negrito(evidência)
  • Itálico(ênfase)
  • Estilo
  • Abreviações
  • Citações
  • Marcações
